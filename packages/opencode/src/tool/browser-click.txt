Click on an element in the browser using a CSS selector or Playwright locator.
Examples: 
- "#submit-btn"
- "button:has-text('Submit')"
- "text=Login"
If the click times out or fails (e.g., element is hidden by an overlay), the tool will return an error with a screenshot attached so you can visually debug why the click failed.
Sometimes Playwright refuses to click an element due to actionability checks (it thinks a transparent div is covering the button). In these edge cases, try passing 'force' set to true to bypass these checks.
