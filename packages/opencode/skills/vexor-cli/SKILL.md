# Semantic Discovery with Vexor

When exploring massive, unfamiliar monorepos, `grep` and `find` often lack context and flood the output with string matches.

Use `vexor` to semantically query implementations, usages, and AST node relationships.

## Concepts
- Understands variable shadowing and scoped references.
- Instantly maps dependencies of a specific function or class across the entire codebase.
- Use to rapidly discover structural patterns instead of blindly grepping.
